# DataReaper Landing Page — Figma AI Prompt (Production-Grade)

---

## Global Design System

**Style:** Cybersecurity SaaS + Futuristic + Minimal Enterprise

**Reference Layout:** Prevalent AI (dark hero → clean sections → editorial typography)

### Colors

* Dark Background: #0B0F1A
* Primary Accent: #6C63FF (gradient: #6C63FF → #8B85FF)
* Secondary Accent: #4FD1C5
* Light Background: #F7F7FB
* Text Primary: #FFFFFF
* Text Secondary: #A0AEC0

### Typography

* Headings: Serif (Playfair Display / Canela style)
* Body: Inter / Satoshi
* Style: High contrast (large elegant headings + minimal body text)

### Layout Rules

* Max Width: 1200px
* Grid: 12-column
* Spacing: Generous whitespace (80–120px vertical spacing)
* Border Radius: 12–20px
* Shadows: Soft + subtle
